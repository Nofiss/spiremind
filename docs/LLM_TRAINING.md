## LLM LoRA/QLoRA Training (Unsloth)

This guide shows how to fine-tune a Llama 3 model with the dataset generated by
`scripts/prepare_llm_dataset.py`. It uses Unsloth for fast LoRA/QLoRA training.

### 1) Generate Dataset

```bash
PYTHONPATH=src python scripts/prepare_llm_dataset.py --lenient
```

Outputs:
- `data/llm/train.jsonl`
- `data/llm/val.jsonl`

### 2) Install LLM Training Stack

```bash
python -m venv .venv
. .venv/Scripts/activate
python -m pip install -U pip
pip install -e ".[llm]"
```

### 3) Train via Unsloth Script

Use the Unsloth training script already in the repo:

```bash
python scripts/llm/train_spire.py \
  --model meta-llama/Meta-Llama-3-8B \
  --train data/llm/train.jsonl \
  --val data/llm/val.jsonl \
  --output spiremind_lora_model
```

Adjust `--model` to match your base model and hardware constraints.

### 4) Inference Sanity Check

Pick a random JSONL line and send the `user` content to the model. Verify the
output is a single valid command like `play 1 0` or `end`.
